Full Transparency

How GameRoute actually works

No marketing fog. Here's exactly what happens, and what doesn't.

1

Ping

GameRoute runs Windows' own ping once a second against Riot's platform endpoints and parses the real round-trip time — the same tool you could run yourself in a terminal.

2

Traceroute

Runs tracert to show every hop between you and the server, highlighting the ones with abnormal latency jumps or timeouts.

3

DNS

Measures a handful of public resolvers (Cloudflare, Google, Quad9, OpenDNS) and lets you switch to the fastest one for your connection — a standard Windows network setting, changed with your explicit confirmation.

4

QoS

Creates a real Windows QoS policy (DSCP tagging) for League's traffic. This only affects how your machine and your own router prioritize packets — it has no effect on the internet path beyond your network.

5

Process Priority

Sets League of Legends to a higher OS scheduling priority so background apps compete with it less for CPU time — reversible in one click from the same tab.

What GameRoute will never do

  • No VPN. GameRoute never creates a virtual network adapter or tunnels your traffic anywhere.

  • No traffic rerouting. Every packet your game sends still goes exactly where your ISP would normally send it.

  • No private backbone. GameRoute doesn't operate (or claim to operate) a relay network. That's genuinely different infrastructure that a local Windows utility can't replicate — so we don't pretend to.

  • No silent changes. Every optimization is behind its own confirmation dialog that states exactly what it will change, before it changes it.

GameRoute never reads or modifies League of Legends' memory, files, or network traffic, and doesn't hook into the game process. It only uses standard Windows features (ping, tracert, DNS settings, QoS, process priority) that any user could apply manually through Windows' own tools. That said, GameRoute is not made or reviewed by Riot Games, so we can't speak for Riot's official stance — if you're risk-averse, stick to read-only features (ping, traceroute, statistics) and skip the optimizer actions.

Your traffic takes exactly the same path your ISP would normally send it on. GameRoute only measures that path (ping, traceroute) and adjusts local Windows settings (DNS server choice, QoS tagging, process priority).

GameRoute runs as an ordinary, independent user-mode application — it doesn't inject code into League of Legends, load a kernel driver, or interact with Vanguard in any way. It runs entirely separately, the same way a ping utility or Task Manager would. Privacy

Your data never leaves your PC

GameRoute was built by someone who was tired of "optimization" tools that quietly phone home. So this one doesn't.

  • Everything stays on your PC

    Settings, ping history and logs are stored under %USERPROFILE%\.gameroute — plain files you can open, back up, or delete yourself.

  • No telemetry

    GameRoute doesn't send usage analytics, crash reports, or any other data about you or your PC anywhere.

  • No account required

    Nothing to sign up for, no email, no login screen. Install it and it works.

  • No data collection

    The only network calls GameRoute makes are the diagnostics themselves (ping/traceroute/DNS to public endpoints) — the feature is the traffic, not a side channel.
